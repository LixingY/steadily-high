# RegExp 最佳实践

### 1. 整数

```js
/^-?[0-9]\d*$/
```

### 2. 正整数

```js
/^[1-9]\d*$/
```

### 3. 正数

```js
/^0\.\d+$|^[1-9]+(\.\d+)?$|[1-9]\d$/
```

### 4. 0-1 之间的数字

```js
/^(0(\.\d+)?|1)$/
```

### 5. 只允许输入字母、-、数字

```js
/[`~!@#$%^&*()_\+=<>?:"{}|,.\/;'\\[\]·~！@#￥%……&*（）——\+={}|《》？：“”【】、；‘'，。、]/
```
